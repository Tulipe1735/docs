## Using AI Assistants When You're Stuck

AI tools like ChatGPT and Claude are genuinely useful learning companions for Next.js. They're especially good at explaining error messages, suggesting fixes for small bugs, and generating code examples you can study. The key is knowing how to ask.

<Tip>
  Always tell the AI which version of Next.js you're using and whether you're using the App Router. Next.js changed significantly between versions, and a vague question might get you an outdated answer.
</Tip>

### Effective Prompt Patterns

Here are prompt templates that get reliable, course-relevant answers:

**For understanding a concept:**
```text
I'm learning Next.js 14 with the App Router. Can you explain how [concept] works 
in simple terms, with a short code example using TypeScript?
```

**For debugging an error:**
```text
I'm using Next.js 14 App Router with TypeScript. I'm getting this error:

[paste your full error message here]

Here's the relevant code:

[paste your code here]

What's causing this and how do I fix it?
```

**For understanding what code does:**
```text
I'm a beginner learning Next.js. Can you walk me through what this code does, 
line by line, in plain English?

[paste your code here]
```

**For generating an example:**
```text
Show me a simple Next.js 14 App Router example of [feature] using TypeScript 
and Tailwind CSS. Keep it beginner-friendly.
```

<Warning>
  AI assistants can be confidently wrong, especially about recent framework changes. Always cross-reference AI-generated code with the official Next.js docs before trusting it in your project.
</Warning>

### Tips for Getting Good Help

When posting a question in any community forum, include:

1. **Your Next.js version** — run `npx next --version` to check
2. **The full error message** — copy it exactly, don't paraphrase
3. **The relevant code** — a minimal snippet that reproduces the issue
4. **What you've already tried** — shows you've put in effort and helps people give better answers

Next.js ships updates regularly. Here are the best ways to keep up with what's changing:
