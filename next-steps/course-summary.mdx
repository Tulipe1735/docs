## What You've Learned

Over the course of these lessons, you built a solid foundation in modern full-stack web development with Next.js 14 and the App Router. Here's everything you accomplished:

- **Set up a professional Next.js development environment** — Node.js, VS Code, extensions, and project scaffolding with `create-next-app`
- **Understand file-based routing and the App Router** — folder structure, nested layouts, and how Next.js turns files into pages automatically
- **Build React components with Server and Client Component patterns** — knowing when to use each, and how they work together in the same app
- **Fetch and display data with async/await** — loading data directly in Server Components without extra boilerplate
- **Create API routes for backend functionality** — building your own endpoints inside the `app/api` directory using Route Handlers
- **Handle errors, loading states, and 404 pages** — using `error.tsx`, `loading.tsx`, and `not-found.tsx` to build resilient user experiences
- **Optimize images and fonts with built-in Next.js features** — leveraging `next/image` and `next/font` for performance without extra configuration
- **Manage environment variables securely** — keeping secrets out of your source code using `.env.local` and Vercel's environment settings
- **Use Git, GitHub, and deploy to Vercel** — taking your project from local development to a publicly accessible URL
- **Apply AI assistants and code quality tools to your workflow** — using tools like GitHub Copilot, ESLint, and Prettier to write better code faster
